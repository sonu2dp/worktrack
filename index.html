Your data auto-saves to Google Drive every time you make a change. Access from any device.

• Upload: Force save current data to Drive now

⚙️ Setup Instructions
Step 1: Go to console.cloud.google.com
Step 2: Create project → Enable Google Drive API
Step 3: Create credentials → OAuth 2.0 Client ID
Step 4: Type: Web Application
Step 5: Add your GitHub Pages URL to Authorized origins
